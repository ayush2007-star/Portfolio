# Ayush Tripathi - Portfolio

A modern, creative portfolio website showcasing skills in Machine Learning, Web Development, and more.

## Features

- **Responsive Design**: Works perfectly on all devices
- **Smooth Animations**: CSS animations and JavaScript interactions
- **Modern UI**: Clean, professional design with gradient backgrounds
- **Interactive Elements**: Hover effects, scroll animations, and more
- **Contact Form**: Functional contact form with validation
- **Dark Mode**: Toggle between light and dark themes
- **Mobile Menu**: Responsive navigation for mobile devices

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with animations
- **JavaScript**: Interactive functionality
- **Font Awesome**: Icons
- **Google Fonts**: Poppins font family

## Sections

1. **Hero**: Introduction with animated text and call-to-action buttons
2. **About**: Personal information with stats and photo
3. **Skills**: Technical skills with progress bars
4. **Projects**: Showcase of key projects
5. **Contact**: Contact form and information
6. **Footer**: Social links and additional navigation

## Customization

### Adding Your Information

1. **Personal Details**: Update name, bio, and contact information in `index.html`
2. **Skills**: Modify the skills section with your expertise levels
3. **Projects**: Add your project details with descriptions and links
4. **Social Links**: Update social media links in the footer

### Styling Changes

- Colors can be modified in the CSS variables at the top of `style.css`
- Fonts can be changed by updating the Google Fonts link
- Animations can be adjusted in the keyframes sections

### Adding Images

1. Replace `img/WhatsApp Image 2026-03-26 at 12.00.57 AM.jpeg` with your photo
2. Add project screenshots to the `img/` folder
3. Update image paths in the HTML accordingly
